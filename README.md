# SafeHer - Women's Safety Platform (Frontend)

[Live Demo](https://safe-her-bd.vercel.app/)

SafeHer is a web-based platform designed to enhance women's safety in Bangladesh. The application provides real-time safety dashboards, interactive maps, and SOS alerts to help users navigate urban environments safely. SafeHer connects users with community resources, safety tips, and local support systems to ensure secure travel and immediate assistance when needed.

---

## Features

- **Safety Dashboard:** Overview of personal safety data, alerts, and safety trends.
- **Interactive Safety Map:** Visualizes safe and unsafe zones in real-time.
- **SOS Modal:** Quickly send distress alerts to local authorities or trusted contacts.
- **Community Section:** Connect with nearby safe spaces and volunteer support.
- **Learning Hub:** Access safety guidelines, tips, and educational resources.
- **Gig Marketplace:** Find verified service providers for personal security.
- **Responsive Design:** Fully mobile-friendly, designed with Tailwind CSS and React.

---

## Tech Stack

- **Frontend:** React, JSX
- **Styling:** Tailwind CSS
- **Build Tool:** Vite
- **Deployment:** Vercel
- **Icons:** Font Awesome
